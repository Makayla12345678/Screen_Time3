# Tech Context

## Technologies Used
- HTML5
- CSS3 (Flexbox, CSS Variables, Transitions)
- JavaScript (ES6+)
- LocalStorage API

## Development Setup
- No build process required.
- Edit `index.html` directly.
- Open in any modern web browser to run.

## Technical Constraints
- Requires browser with `localStorage` support.
- No backend sync (data is local to the device/browser).
